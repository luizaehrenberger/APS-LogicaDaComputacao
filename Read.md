# LINHA — DSL para robô seguidor de linha

**Objetivo:** A LINHA é uma linguagem de domínio específico pensada para a APS de Lógica da Computação. O contexto é simples e concreto: controlar um robô seguidor de linha com dois sensores ópticos (esquerdo/direito) e um sensor de distância, além de dois motores. A linguagem é mínima e didática, ideal para demonstrar o pipeline de compilação (léxico → sintático → geração de código/assembly para uma VM).
